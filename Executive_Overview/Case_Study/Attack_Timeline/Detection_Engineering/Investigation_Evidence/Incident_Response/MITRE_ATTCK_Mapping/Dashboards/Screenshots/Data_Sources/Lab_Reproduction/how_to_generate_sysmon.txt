# Lab Reproduction

1. Install Sysmon (registry rules).
2. Run: `reg add HKCU\...\Run /v FakeUpdater /t REG_SZ /d notepad.exe`.
3. Reboot/logon; query Splunk.
4. Cleanup: `reg delete HKCU\Run\FakeUpdater /f`. [file:1]

## Lab Reproduction Notes

Persistence telemetry was generated using legitimate Windows activity and controlled registry modifications with benign binaries.

This approach ensures realistic detection engineering without introducing malware.
